Crowdsourcing Conference 2012
=============================

http://www.conferenciacrowdsourcing.com.br/

Install
-------

Clone the project
  `git clone git://github.com/softa/ccs12.git`

Enter in your app folder
  `cd ccs12`

Install dependencies
  `bundle install`

Run the tests
  `rake spec`

Start the Server
  `rails server`

Setting the variables
---------------------
  for heroku
    `thor -T`
